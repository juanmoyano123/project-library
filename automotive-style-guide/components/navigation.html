Navigation Design Guidelines

  • Header height: 80px on desktop, 64px on mobile
  • Fixed positioning: Header stays fixed at top with scroll behavior
  • Menu overlay: Full-screen dark overlay (z-index: 200)
  • Animation timing: 0.6s cubic-bezier(0.38, 0.005, 0.215, 1)
  • Staggered entry: Menu items animate with 0.1s delay between each
  • Keyboard support: ESC key closes menu, tab navigation works
  • ARIA labels: All interactive elements have proper labels
  • Touch targets: Minimum 44px height for mobile
