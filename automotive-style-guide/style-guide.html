Automotive Design System

A precision-engineered design system inspired by the automotive industry. This guide documents all visual patterns, components, and implementation details for creating consistent, high-quality digital experiences.

Design Principles

Precision

Every element is carefully measured and aligned. Clean lines, exact spacing, and mathematical relationships create visual harmony.

Confidence

Bold typography and high-contrast design communicate strength and authority without unnecessary decoration.

Clarity

Information hierarchy is always clear. Users never question what to do next or where to find information.

Performance

Optimized for speed and efficiency. Smooth animations, fast loading, and responsive interactions.

Typography

Two font families create a strong typographic hierarchy: ABCDiatype for readability and SuisseIntlCond for technical precision.

Font Families

ABCDiatype

Primary typeface for headings, body text, and all major content. Available in Regular (400) and Bold (700) weights.

SUISSEINTLCOND

Condensed typeface for labels, buttons, and UI elements. Creates technical precision.

Type Scale

Heading 1 - 83.33px / 400

Heading 2 - 48px / 700

Heading 3 - 24px / 700

Heading 4 - 18px / 700

Body Large - 30px / 400 / -0.6px

Body Default - 18px / 400

Label Text - 14px / UPPERCASE / 0.5px

CSS Implementation

/* Typography Variables */ --font-primary: 'ABCDiatype', -apple-system, system-ui, sans-serif; --font-condensed: 'SuisseIntlCond', -apple-system, system-ui, sans-serif; --text-xs: 12px; --text-sm: 14px; --text-base: 18px; --text-lg: 24px; --text-xl: 30px; --text-2xl: 48px; --text-3xl: 83.3333px; --weight-normal: 400; --weight-bold: 700; /* Heading Styles */ h1 { font-size: clamp(48px, 8vw, var(--text-3xl)); font-weight: var(--weight-normal); letter-spacing: -3.33px; line-height: 1; } h2 { font-size: clamp(32px, 5vw, var(--text-2xl)); font-weight: var(--weight-bold); letter-spacing: -1px; }

Usage Guidelines

  • xs (8px): Internal padding for small elements, icon gaps
  • sm (16px): Text spacing, button padding, card gutters
  • md (24px): Component spacing, section gaps
  • lg (32px): Large component padding, feature spacing
  • xl (48px): Section padding (mobile), major dividers
  • 2xl (64px): Section padding (tablet), hero spacing
  • 3xl (96px): Section padding (desktop), major sections
